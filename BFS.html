Breadth-First Search

PSEUDOCODE

BFS(graph, start):
  queue = [start]
  visited = set()
  
  while queue is not empty:
    current = queue.dequeue()
    
    if current not in visited:
      visited.add(current)
      process current
      
      for neighbor in graph[current]:
        if neighbor not in visited:
          queue.enqueue(neighbor)

ALGORITHM EXPLANATION

BFS explores a graph level by level, starting from the root node. It uses a queue to keep track of nodes to visit, ensuring all nodes at the current depth are explored before moving to the next level. Perfect for finding shortest paths in unweighted graphs.
